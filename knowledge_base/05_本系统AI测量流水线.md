# AI 辅助测量流水线（技术为辅）

> **阅读顺序建议**：先掌握 02–04、13 的物理与误差，再读本节。AI 不改变 σ 的计算公式。

## 作用边界
- **输入**：衍射图像（含刻度尺）；
- **输出**：H₀、Δx₁、Δx₂（及像素值）、标尺 R²、标注图、光强剖面；
- **不输出**：σ（由数据处理页根据物理公式计算）。

## 五步流程（对应 Web 进度）
| 步骤 | 模块 | 物理相关说明 |
|------|------|----------------|
| 1 | 图像预处理、1D 剖面 | 将 2D 图样化为 I(y) |
| 2 | 标尺 OCR | 像素↔cm；异常时 R² 低 |
| 3 | SIREN 拟合 I(y) | 连续光强曲线 |
| 4 | PINNs 约束 + 牛顿求峰 | 亚像素极值 |
| 5 | 可视化 | 人机协同验证用 |

## 双通路峰值检测
- A：find_peaks + 平滑；
- B：SIREN + 牛顿迭代；
左右间距比异常时融合/回退。

## 学生必须做的验证
1. 看**标注图**：峰线是否在亮纹中心；
2. 看 **R²**；
3. 与手测 Δx、H₀ 对比，讨论差异属于随机还是系统。
