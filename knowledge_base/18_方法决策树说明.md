# 方法决策树（AI 读数选型 · 为辅）

## 物理前提
任务本质是：**一维光强剖面上找三个极大值并换算几何量**——不是 2D 图像分类。

## 决策逻辑
1. 有大量像素级标注？否 → 排除监督分割。  
2. 是否 1D 剖面？是 → 排除通用 2D 检测器为主方案。  
3. 是否需要亚像素可微极值？是 → 倾向连续隐式表示 + 求导。  
4. 结论：**SIREN + 物理约束损失 + 牛顿 refine**（见 06–08）。
