# 实验概述与物理框架

> 文件名保留历史编号；内容为**物理实验总览**，非竞赛专用说明。

## 实验名称与目的
**基于激光衍射的液体表面张力系数测量**（He–Ne 激光，真空波长 λ₀ = 632.8 nm）。

**物理目的**：通过毛细波（表面波）在液体表面形成的周期结构，使入射激光产生可观测的衍射图样；由条纹几何关系得到波数 k，再结合 Kelvin 色散关系由激励频率 f 反演表面张力系数 σ。

**能力目标**：
1. 理解毛细波色散、夫琅禾费衍射与几何光学推导链；
2. 掌握 H₀、Δx、δ、k、σ 各量的定义与单位；
3. 完成传统读数与（可选）AI 辅助读数对比及误差分析；
4. 会调节光路、判断图样是否可用于分析。

## 实验装置与光路要点（概念）
| 部分 | 作用 | 常见失误 |
|------|------|----------|
| He–Ne 激光器 | 提供相干、单色照明 | 波长记错单位；光束未扩束 |
| 扩束/准直 | 照明面积与平行度 | 照斑过小或发散角大 |
| 液体槽+振源 | 激发频率 f 可调的毛细波 | f 不稳定；液面振动杂模 |
| 屏或相机 | 记录衍射图+刻度尺 | 过曝、标尺反光、视差 |
| 刻度尺 | 将像素位置映射为 cm | 10/20/30 cm 未入镜、倾斜 |

## 测量量与计算分工
| 物理量 | 含义 | 主要来源 |
|--------|------|----------|
| H₀ | 零级衍射峰在刻度尺上的高度读数 | 图像读数（人工或 AI 辅助） |
| Δx | 零级到 ±1 级条纹间距 | 同上 |
| h | 液面相对参考面下降量 | **独立测量**（米尺/标定） |
| L | 液面到记录平面距离 | **独立测量** |
| f | 激励频率 | 信号源读数 |
| ρ | 液体密度 | 查表或实测（注明温度） |
| σ | 表面张力系数 | **数据处理页公式计算**，非 AI 直接输出 |

## Web 系统三页（辅助工具）
1. **图像采集页**：上传衍射图，得到 H₀、Δx 及标注图、剖面、R²；
2. **数据处理页**：代入 f、ρ、L、h 等，按 Kelvin 与几何公式算 σ；
3. **智慧星助手**：基于本知识库答疑（物理优先）。

## 核心公式链（一览）
$$H = H_0 - h$$
$$\alpha = \arctan\frac{H}{L},\quad \beta = \arctan\frac{\Delta x + H}{L},\quad \delta = \beta - \alpha$$
$$k = \frac{2\pi}{\lambda_0}\sin\frac{\delta}{2}\left[\sin\left(\alpha+\frac{\delta}{2}\right)+\sin\left(\alpha-\frac{\delta}{2}\right)\right]$$
$$\sigma = \frac{\rho\omega^2}{k^3},\quad \omega = 2\pi f$$
