# 人机协同与 AI 边界

## 原则
**物理实验报告的主体是原理、数据、误差与讨论**；AI 是读数与答疑工具。

## 分工
| 环节 | 学生/人工 | AI 辅助 |
|------|-----------|---------|
| 光路、样品 | ✓ | — |
| 判图样质量 | ✓ | 提示 |
| 读 H₀、Δx | 可 ✓ | 可辅助 |
| 测 h、L、f、ρ | ✓ | — |
| 算 σ、写误差 | ✓ | 不代算 |
| 概念答疑 | ✓ | 智慧星 RAG |

## 报告写作
须写清公式、单位、不确定度；AI 数据须与手测或图样对照，**不得**写“因 AI 输出故正确”。
